# Desarrollo

El código fuente completo (`index.html`, `css/styles.css`, `js/script.js`) está ubicado en la **raíz** del repositorio, no dentro de esta carpeta.

Esta decisión se tomó para que GitHub Pages pueda desplegar el sitio con la configuración más simple ("Deploy from a branch: main / root"), sin necesidad de configurar GitHub Actions.

Ver [Documentacion/README.md](../Documentacion/README.md) para instrucciones de uso, decisiones técnicas y pasos de despliegue.
